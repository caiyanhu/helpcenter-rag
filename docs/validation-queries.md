## 验证方法

对每个查询，记录以下指标：

```
查询: "如何创建账号"
Dense Top5: [创建账号, 数据库管理, 创建数据库, ...]
Hybrid Top5: [创建账号, 数据库管理, 创建数据库, ...]
重叠率: X/5
人工判断: 
  - Dense 相关度: 1-5分
  - Hybrid 相关度: 1-5分
  - 哪个更好: Dense/Hybrid/平手
```

```bash
cd backend
npx tsx scripts/compare-search-standalone.ts
```
